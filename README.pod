=head1 NAME

MUTATE :: The A/V Toolkit for MPlayer

=head1 SYNOPSIS

	Retrieve information about your media collection using mscan:

	mutate mscan ~/Music ~/Videos

	Create FFmpeg movies using mcode:

	mutate mcode dvd://1 -o file.avi -m "-af volume=10"

	Distributed encoding via the web gui...

	mutate daemon

=head1 DESCRIPTION

MUTATE is a set of media tools written in Perl, and for use with MPlayer and MEncoder.
MUTATE also provides a web based graphical interface capable of distributing encoding
resources across multiple MUTATE instances on your private network or over the internet.
